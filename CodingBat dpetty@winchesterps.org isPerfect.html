Write the method boolean isPerfect(long p) that returns true if p is perfect. Following Euclid's formula, p = 2**(n - 1)[2**n - 1] is a perfect number when [2**n - 1] (the nth Mersenne number) is prime. Not all numbers p (e.g. 1, 6, 28, 120, 496, 2016, 8128, 32640, 130816, 523776, 2096128, 8386560, 33550336, 134209536, 536854528, 2147450880, 8589869056, 34359607296, 137438691328, 549755289600, 2199022206976, 8796090925056, 35184367894528, 140737479966720, 562949936644096, 2251799780130816, 9007199187632128, 36028796884746240, 144115187807420416, 576460751766552576, 2305843008139952128, 9223372034707292160) are perfect, so test whether [2**n - 1] is prime! Note: you will need another method with the signature: public boolean isPrime(long n) — add it below the code for isPerfect.
